# Sharkware Gaming — Presentación

Deck de presentación del proyecto **Sharkware Gaming** (e-commerce de hardware gaming).
HTML estático autocontenido con los clips de demo embebidos.

## Ver online
GitHub Pages: una vez activado, queda en `https://<usuario>.github.io/<repo>/`

## Ver localmente
Abrir `index.html` en el navegador.

- **← →** (o click izquierda/derecha): navegar slides
- **F**: pantalla completa
- Los clips de cada slide de video reproducen solos al entrar.
